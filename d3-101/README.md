## Selections
Selections are arrays.

\#foo        // `<any id="foo">`  
foo         // `<foo>`  
.foo        // `<any class="foo">`  
[foo=bar]   // `<any foo="bar">`   
foo bar     // `<foo><bar></foo>`  
foo.bar     // `<foo class="bar">`  
foo\#bar     // `<foo id="bar">`  

Chaining can simplify code.
